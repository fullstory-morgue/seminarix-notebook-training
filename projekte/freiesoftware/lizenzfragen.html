Achten Sie auf die Lizenzbedingungen!

Wie bereits im Vorwort geschrieben, besteht eine der im Vorbereitungsdienst zu leistenden Aufgaben darin, dass sich die künftigen Lehrerinnen und Lehrer eine kritische medienpädagogische Kompetenz erwerben. Die auf dieser CD gesammelten Artikel, die Software und die Materialien sollen dies unterstützen.

Die hier gesammelte Software und die Materialien können allesamt erprobt werden. Damit ist es möglich, sich einen großen Fundus an Software zu erschließen und Maßstäbe für die Qualität der Software zu entwickeln.

Nicht alle Programme sind allerdings ohne Einschränkungen in Schule und Seminar einsetzbar. Dies sind zunächst die Demonstrationsversionen (Demoversion) kommerzieller, proprietärer Software. Diese Programme dürfen Sie erproben, vor dem Einsatz im Unterricht oder zur Unterrichtsvorbereitung und Nachbereitung müssen Sie die Programme aber erwerben.

Eine weitere Kategorie von Software ist sogenannte Shareware. In diese Software können Sie sich ohne Einschränkung einarbeiten. Sie können Sie auch im Unterricht nutzen, wenn es z.B. darum geht, die Programme mit anderen zu vergleichen. Sobald Sie allerdings die Software beruflich, also auch im Unterricht, oder privat regelmäßig nutzen, wird die sogenannte Shareware-Gebühr, die der Programmautor festlegt, fällig.

In Schule und privat kostenfrei nutzbar sind Public-Domain-Programme und Freeware.
Während Public-Domain-Programme kostenfrei ohne jedwede Einschränkung genutzt werden kann können, ist Freeware im Allgemeinen kostenfrei zu nutzen. Der Autor kann aber Beschränkungen der Nutzung festlegen. Bei Freeware sind deshalb die Lizenzbedingungen genau zu prüfen. Schauen Sie sich die Lizenzbedingungen von Programmen wie GrafStat oder MindManager (Schulversion) einmal genau an! Abgespeckte oder sonst wie in der Funktionalität beschränkte Versionen kommerzieller Software ist oftmals Freeware. Was davon dann letztlich in Seminar und Schule eingesetzt wird, kann nur von Qualitätsgesichtspunkten geleitet ausgewählt werden. Lizenzkosten gibt es nicht.

Sind Public-Domain-Programme oder Freeware also die ideale Software für die Nutzung in Schule und Lehrerausbildung? Nein! Denn auch diese Programme sind in einem Punkt genauso eingeschränkt, wie die kommerzielle Software und die Shareware: Ihr Programm-Code ist in der Regel nicht verfügbar. Die Behebung von Fehlern, die Weiterentwicklung und die Nutzung in anderen Programmierprojekten ist nur durch den Autor, der über den Programm-Text verfügt, machbar.

Diese Beschränkung ist bei der auf dieser CD vorhandenen Freien Software (mit OpenSource versehen) nicht vorhanden. Freie Software ist in der Regel auch kostenfrei verfügbar. Die Beschaffung kann aber mit Kosten verbunden sein, wenn z. B. ein Verlag eine Zusammenstellung von freier Software auf DVD/CD anbietet. Bezahlt werden müssen dann die Versandkosten, die Materialkosten und das Entgelt für das Zusammenstellen der Software. Hat man die Software aber einmal beschafft, dann greifen die Lizenzbedingungen! Die sind allerdings etwas so Besonderes, dass man hier oftmals davon spricht, dass die Software nicht mit einem Copyright versehen ist, sondern mit einem Copyleft! Folgende Rechte und Pflichten hat der Nutzer Freier Software:
  • Er kann die Software ohne Einschränkung für jeden Zweck nutzen.
  • Er kann die Software ohne Einschränkung verteilen; er kann dafür sogar einen Preis verlangen.
  • Er kann die Software beliebig verändern. Damit dieses Recht genutzt werden kann, muss bei Freier Software immer der Quelltext mitgeliefert werden. Deshalb wird auch oft von OpenSource-Software gesprochen, die aber keine Freie Software sein muss.
  • Der Nutzer hat die Pflicht, jede Weiterentwicklung des Programms unter den gleichen Lizenzbedingungen weiterzugeben. Er muss sie aber nicht weitergeben.
Die bekannteste Lizenz für freie Software ist dievon Richard Stallman entwickelte GNU Public Licence, die GPL. Ich empfehle, diese Lizenz zu studieren.
Die GPL ist so attraktiv, dass eine Fülle von riesigen Software-Projekten unter dieser Lizenz entwickelt und vertrieben wird. Das wichtigste Projekt ist sicherlich das Betriebssystem GNU/Linux. Einen Teil der unter dieser Lizenz stehenden Software finden Sie auf dieser CD. Diese Software können Sie also beruflich und privat beliebig nutzen. Sie können Sie aber auch ohne Beschränkungen an Ihre Kolleginnen und Kollegen und an Ihre Schülerinnen und Schüler weitergeben. Wenn Sie Wünsche zur Weiterentwicklung haben, können Sie sich in der Regel an die Entwicklergemeinschaft oder eine Kollegin oder einen Kollegen wenden, der programmieren kann. Seminar und Schule könnten für die Weiterentwicklung aber auch statt einer kommerziellen Software Programmierleistung einkaufen. Die Weiterentwicklung stünde dann ohne weitere Kosten wieder allen Schulen und Seminaren zur Verfügung. Ob und wie Sie diese Software nutzen, hängt nur davon ab, ob sie Ihren Ansprüchen genügt. Die Nutzung kommerzieller Software im Unterricht bedarf deshalb einer gewichtigen Begründung: Es müssen wesentliche Gründe vorhanden sein, die die Kosten und die vorhandenen Einschränkungen ausgleichen.

Begründet werden muss deshalb im Grundsatz auch die Wahl des eingesetzten Betriebssystems: Welche gewichtigen Gründe rechtfertigen es, dass ein kommerzielles Betriebssystem wie Windows oder das MAC-OS eingesetzt wird und nicht das freie Betriebssystem GNU/Linux oder ein anderes freies Betriebssystem wie BSD o.ä. .

Da Sie in Schule und Seminar aus verschiedenen Gründen häufig auf das Betriebssystem Windows stoßen werden, sind auf dieser CD nur unter Windows arbeitende Programme versammelt. Einige davon sind in gleicher Weise für GNU/Linux vorhanden, z. B. die Büro-Software (das „Office-Paket“) OpenOffice, der Webbrowser Firefox oder die Mindmapping-Software FreeMind. Sie können also schon erproben, ob Sie diese Programme verwenden können. Bei einem Wechsel des Betriebssystems können Sie die mit diesen Programmen erstellten Arbeitsergebnisse ohne Probleme weiter nutzen.
